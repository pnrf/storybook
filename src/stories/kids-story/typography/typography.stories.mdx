<a name="fonts"></a>
## Шрифты

В дизайне применяются следующие шрифты:

- **Montserrat** от Google Fonts ([официальная страница](https://fonts.google.com/specimen/Montserrat))

Семейство шрифтов размещаем в папке проекта: `src/vendor/fonts`. В ней же создаем файл `fonts.css`:
```css
// fonts.css

@font-face {
  font-family: 'Montserrat';
  font-style:  normal;
  font-weight: 900;
  src: url("./Montserrat/Montserrat-Black.ttf") format("truetype");
}

...и т.д. для каждого font-weight: 900, 800, 700, 600, 500, 400, 300, 200, 100
```
Шрифты подключаются в `src/index.css` после `normalize.css`:
```css
// index.css

@import url(./vendor/normalize.css);
@import url(./vendor/fonts/fonts.css);

#root {
  font-family: 'Montserrat', Arial, sans-serif;
}
```
Поскольку шрифт только один, то использовать css-переменные не нужно. Если бы шрифтов было несколько, можно было бы задать переменные. Например:
```css
--font-family: Montserrat, Roboto, Arial, sans-serif;
--title-font-family: Roboto, Montserrat, Arial, sans-serif;
```
