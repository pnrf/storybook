# Логотип

- Для логотипа применяется `svg` формат файла;
- Онлайн-сервис для оптимизации `svg`: [https://jakearchibald.github.io/svgomg/](https://jakearchibald.github.io/svgomg/);
- Онлайн-сервис для оптимизации `jpg, png`: [https://tinypng.com](https://tinypng.com);

------

### Примеры использования
- в Header и в Footer десктопной версии сайта:
<LogoLarge />

- в Header и в Footer мобильной версии сайта:
<LogoSmall />
